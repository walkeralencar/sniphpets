inModule  [line 46]

void inModule( string $pModuleName  )

Retorna o caminho do Módulo.

Retorna o caminho do Módulo.
Será melhor utilizada se trabalhar com projetos tratando cada módulo em uma pasta, para quem está pensando em estudar MVC, será de boa ajuda, para o início dos estudos.
Partindo da estrutura de pastas:

  1.  Clientes....: /projeto/modules/cliente
  2.  Fornecedores: /projeto/modules/fornecedor
  3.  Produtos....: /projeto/modules/produto
Ex.:
  1.  <?php
  2.  //o inModule('cliente') retorna: '/projeto/modules/cliente/'
  3.  include_once ( inModule( 'cliente' ) . 'view/list.php' );
  4.  ?>

 Parameters:
string   $pModuleName:  Nome do Módulo
