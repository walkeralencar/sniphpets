string ArrayToOption( array $pItem, [mixed $pAtual = 0]  )

Gera os options, com base em um array informado e em conformidade com o W3C - XHTML 1.

Gera os options, com base em um array informado e em conformidade com o W3C - XHTML 1.0

  1.  <?php
  2.  $aTeste = array(1 => 'um',2 => 'dois',5 => 'cinco');
  3.  echo "<select name=\"teste\">".ArrayToOption($aTeste,2)."</select>";
  4.  ?>
Resultado:
  1.   <select name="teste">
  2.     <option value="1" >um</option>
  3.     <option value="2" selected="selected" >dois</option>
  4.     <option value="5" >cinco</option>
  5.   </select>

 Parameters:
array   $pItem:  Lista com as opções em um array.
mixed   $pAtual:  Key do item que deve estar marcado com: selected.

geraPaginacao  [line 49]

string geraPaginacao( string $pURL, int $pTotal, int $pLimite, [int $pCurrentPage = 1]  )

Gera paginacao, de lista de registros de tabela por exemplo.

Gera paginacao, de lista de registros de tabela por exemplo.
Recomenda-se o uso deste CSS( paginacao.css ) na página. Você pode personalizá-lo como bem entender.
Modo de uso:

  1.  <?php
  2.  echo geraPaginacao('clientes.php', 6, 4, (checkGET('np')?$_GET['np']:1) );
  3.  ?>
Resultado:
  1.    <div class="paginacao">
  2.        <ul>
  3.            <li><a href="clientes.php?np=1" class="marcado">1</a></li>
  4.            <li><a href="clientes.php?np=2" >2</a></li>
  5.        </ul>
  6.    </div>

 Parameters:
string   $pURL:  Endereco do link a ser gerado.
int   $pTotal:  Qtd total de Registros.
int   $pLimite:  Limite de Registros por página.
int   $pCurrentPage:  Página Atual, gerado o valor em $_GET['np'].
