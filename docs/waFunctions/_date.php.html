chooseDate  [line 107]

string chooseDate( string $pName, [string $timestamp = &quot;&quot;]  )

Gera os Selects de Dia, Mês e Ano.

Gera os Selects de Dia, Mês e Ano. Um ao lado do outro, e deixa selecionado a data que for informada, desde que essa data seja informada no formato americano: AAAA-MM-DD.
Ex:

  1.   <?php echo chooseDate('dtNascimento'); ?> //em HTML
Ao submeter o formulário tratar como um array, onde:
  1. $var['d'] - contém o dia.
  2. $var['m'] - contém o Mês.
  3. $var['y'] - contém o Ano.
    1.  <?php
    2.  $dtNasc = $_POST['dtNascimento'];
    3.  if ( ! checkdate( $dtNasc[ 'm' ], $dtNasc[ 'd' ], $dtNasc[ 'y' ] ) ) {
    4.      echo printf('Data de Nascimento[%s/%s/%s] é inválida!',$dtNasc[ 'd' ],$dtNasc[ 'm' ],$dtNasc[ 'y' ]);
    5.  }
    6.  ?>

 Parameters:
string   $pName:  Nome do elemento <select>
string   $timestamp:  Data em formato americano: YYYY-MM-DD
